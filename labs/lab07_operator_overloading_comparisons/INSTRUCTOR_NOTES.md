# Instructor Notes: Lab 7

## Primary Goals

Students should see that Python classes can define behavior for built-in operations.

This lab prepares students for later sorting/searching discussions.

## Suggested Live Modifications

- Add `__add__`
- Print the fastest time
- Add a new race result
- Change formatting to MM:SS
- Compare two manually created objects

## Watch For

Students may not understand why `sorted(times)` works only after `__lt__` is implemented.
