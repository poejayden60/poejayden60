# Lab 1: Environment Setup and GitHub Workflow

## Goals

By the end of this lab, you should be able to:

1. Create or access a GitHub account.
2. Fork the CPSC 250L lab repository.
3. Clone your fork into PyCharm.
4. Configure a Python interpreter in PyCharm.
5. Install the required Python packages.
6. Edit and run a Python file.
7. Commit your changes.
8. Push your changes to GitHub.
9. Show your instructor your pushed work on GitHub.
