# Instructor Notes: Lab 8

## Primary Goals

Lab 8 is the Week 2 integrated OOP challenge.

It should reinforce:
- individual objects
- lists of objects
- object methods
- simple searching
- sorting using comparisons
- branch/merge workflow

## Suggested Live Modifications

- Add a new book
- Change the low-stock threshold
- Search for another author
- Sort by price instead of title
- Sell copies of a book and reprint inventory

## Watch For

Students may struggle to distinguish:
- methods that belong inside `Book`
- functions that operate on a list of books

This is a useful design discussion.
